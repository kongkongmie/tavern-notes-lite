# Tavern Notes Lite 作者中文注释

## 0.2.0

### 大更新

- 重做结构，酒馆笔记 Full / Lite 共用核心。
- UI界面大更新，修复上版本问题，更多预览空间：
  - 笔记卡片重新排版，正文预览区域更大；点击卡片可以查看完整内容，并直接使用复制、重新输入、分享、编辑和删除功能。
  - 顶部按钮统一为图标，并会按照面板宽度自动展开或收进“更多”菜单；主题入口已经移动到右上角工具栏。
- 整楼摘录增加“排除标签”设置，可以去掉正文内不想要的内容。
- 新增“版本与更新”页面，可以检查新版本，并同时查看默认更新日志与作者补充说明。
- 已移除问题较多的 Secret Files 内置主题，并修复多种主题下的颜色、浮层和响应式布局问题。（后期会重做此主题）
- 关闭“记录输入”现在只会停止之后的自动记录，不会再隐藏已经保存的 User 输入或手动创建的灵感笔记。
- Lite 仍然把笔记保存在当前浏览器的 IndexedDB 中，不需要安装 Server Plugin。卸载扩展、清理浏览器数据或更换设备前，请先导出 JSON 备份。
